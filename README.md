# Docker Service Stack

Author: Urmate

A Compose project with an Nginx frontend, Python backend, health route, and production-like wiring.

## Stack

`Docker`, `Nginx`, `Python`

## Theme

- Name: Harbor Stack
- Primary: #118ab2
- Accent: #06d6a0
- Motion: 240ms container rise, network sweep

Theme tokens are also stored in `.urmate/theme.json` for Urmate Builder.
